## Tecnologías Utilizadas
Este proyecto fue construido utilizando:

* **HTML5:** Estructura semántica.
* **CSS3:** Diseño responsivo (Flexbox & Grid), Variables CSS y Modo Oscuro/Claro.
* **JavaScript:** Manipulación del DOM y lógica interactiva.
* **Git & GitHub:** Control de versiones y despliegue.

## Estructura del Proyecto
El código está organizado siguiendo estándares profesionales:
* `/assets`: Contiene todos los recursos (imágenes, estilos, scripts).
* `index.html`: El punto de entrada principal.
